# Library Management System (C++)

## Project Overview

The Library Management System is a simple console-based application developed in C++. This project was created as part of the Programming Fundamentals course. It helps manage library books by allowing users to add books, view available books, borrow books and return borrowed books.

## Features

* Add new books to the library
* Display all available books
* Borrow a book using its ID
* Return a borrowed book
* Menu-driven user interface
* Book availability tracking

## Technologies Used

* C++
* Structures
* Arrays
* Functions
* Loops and Conditional Statements

## How the System Works

1. The user selects an option from the menu.
2. Books can be added with an ID, title, and author name.
3. The system displays all stored books and their availability status.
4. Users can borrow a book if it is available.
5. Borrowed books can be returned using their ID.
